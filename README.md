pyqtkeybind
===============================

version number: 0.0.1
author: Arun Mahapatra

Overview
--------

Cross platform global keybindings for PyQt5

Installation / Usage
--------------------

To install use pip:

    $ pip install pyqtkeybind


Or clone the repo:

    $ git clone https://github.com/codito/pyqtkeybind.git
    $ python setup.py install
